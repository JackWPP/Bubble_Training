# 06 — 训练配置细节

## 一、最优模型完整配置

### 1.1 训练命令

```bash
python segmentation/scripts/train_seg.py \
    --model segmentation/configs/models/bubble_yolo11s_seg.yaml \
    --cfg segmentation/configs/train/seg_full_sgd_zeroaug.yaml \
    --name bubble_seg_dice01 \
    --device 0 \
    --dice 0.1
```

### 1.2 模型配置

`bubble_yolo11s_seg.yaml`:
```yaml
nc: 1
scale: s
scales:
  s: [0.50, 0.50, 1024]

backbone:  # 标准 YOLO11s (10 层)
head:      # FPN + Segment (23 层)

Segment: [[16, 19, 22], 1, Segment, [nc, 32, 256]]
# nc=1 (单类), nm=32 (原型掩码), npr=256 (隐藏通道)
```

## 二、Dice Loss 实现

文件: `segmentation/scripts/dice_loss.py`

通过 monkey-patch `v8SegmentationLoss.single_mask_loss` 注入混合损失：

```python
loss_mask = (1 - w) * BCE + w * (1 - Dice)
```

其中 Dice 系数为平滑版本:
$$Dice = \frac{2|P \cap G| + 1}{|P| + |G| + 1}$$

## 五、自定义模块注册

训练前需调用 `register_bubble_modules()` 注册 16 个自定义模块（即使最终模型不使用）:

```python
from ultralytics_custom import register_bubble_modules
register_bubble_modules()
```

这确保了 YAML 解析器能识别自定义模块名称。当前最优模型不使用任何自定义模块。
