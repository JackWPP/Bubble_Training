# 00 — 术语标准化对照表

本文档建立工程实验记录中的非正式用语（"黑话"）与毕业论文中应使用的规范学术表达之间的映射关系。

## 使用原则

1. **首次出现**：每个术语首次在正文中出现时，应给出完整中文名称，并在括号内标注英文全称。例如："局部对比度细化模块（Local Contrast Refinement Module，简称 LCRefine）"。
2. **后续引用**：此后可使用中文简称或英文缩写。
3. **统一性**：全文同一概念使用同一表达，不得混用。
