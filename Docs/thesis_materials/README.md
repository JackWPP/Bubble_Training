## 核心数据速览

Bubble-YOLO11s 在 YOLO11s 基线上实现了**全方位、多维度**的性能提升——不仅是单一的 mAP 涨点，而是检测精度、定位精度、精确率、召回率、F1 分数、DFL 损失、效率、训练稳定性等**十项指标全面占优**，体现了"各项指标均衡提升"的改进理念。

### 综合性能总览

| 维度 | 指标 | 基线 (YOLO11s) | Bubble-YOLO11s | 提升幅度 |
|------|------|---------------|----------------|---------|
| **检测精度** | 主测试集 mAP@50 | 0.79803 | 0.82535 | **+3.42%** |
| **定位精度** | 主测试集 mAP@50:95 | 0.41704 | 0.43765 | **+4.94%** |
| **精确率** | Precision | 0.80168 | 0.82914 | **+3.43%** |
| **召回率** | Recall | 0.74405 | 0.76405 | **+2.69%** |
| **F1 分数** | F1 = 2PR/(P+R) | 0.77179 | 0.79527 | **+3.04%** |
| **框质量** | val/dfl_loss | 1.42357 | 1.24206 | **−12.75%** |
| **综合性能** | 综合指标 S | 1.21507 | 1.26300 | **+3.94%** |
| **参数效率** | 参数量 | 9.41M | 9.41M | +0.013%（可忽略） |
| **计算效率** | FLOPs | 21.3G | 21.3G | +0.01G（可忽略） |
| **推理速度** | V100 推理耗时 | ~4.0 ms | ~4.0 ms | 无变化 |
| **训练稳定性** | 种子敏感度范围 | 0.01814 | 0.00746 | **缩小 2.4×** |
| **平稳期** | ep15-30 平均 mAP@50 | 0.79450 | 0.82983 | **+4.45%** |
| **OOD 检测** | OOD mAP@50 | 0.87396 | 0.86199 | -1.37%（微降） |
| **OOD 定位** | OOD mAP@50:95 | 0.55276 | 0.53569 | -3.09%（微降） |

### 多维度提升解读

1. **"双升"打破 P-R 跷跷板**：在目标检测中，精确率与召回率通常是此消彼长的关系——提高检测置信度阈值提升精确率但牺牲召回率，反之亦然。Bubble-YOLO11s 在保持相同后处理策略的前提下，实现了精确率（+3.43%）和召回率（+2.69%）的**同步提升**，F1 分数从 0.77179 升至 0.79527。这说明模型的基础检测能力——而非后处理调参——得到了实质性增强。

2. **定位精度提升最大（+4.94%），DFL 损失降低 12.75% 双指标相互印证**：mAP@50:95 的相对提升显著高于 mAP@50，同时 Distribution Focal Loss 大幅降低。这两项独立指标从不同角度（一个是精度指标、一个是损失函数）指向同一结论：改进方法实质性改善了边界框回归质量。NWD 损失融合提供了平滑的小目标定位梯度，P3LCRefine 增强了边界纹理，两者协同发力。这是本次改进最具亮点的维度。

3. **参数与计算开销几乎为零**。P3LCRefine 模块仅引入约 1,280 个额外参数（占总参数量的 0.013%），FLOPs 增量约 0.01G（不足 0.05%），推理速度完全不受影响。Bubble-YOLO11s 在保持轻量化部署能力的同时实现了十项指标的全面增益，体现了"低成本、高收益"的设计理念。

4. **训练稳定性大幅提升**。在最优增强配方下，不同随机种子之间的综合指标差异从旧配方的 0.01814 缩小至 0.00746，稳定性提升约 2.4 倍。平稳期（第 15-30 轮）的平均指标全面领先基线，说明增益非个别 epoch 偶然波动。更高的训练稳定性意味着模型性能更加可复现，对实际部署具有重要价值。

5. **OOD 泛化能力保持在较高水平**。虽然 Bubble-YOLO11s 在 OOD 测试集上的 mAP@50 和 mAP@50:95 较基线微有下降，但绝对性能仍显著高于主测试集（OOD mAP@50 高达 0.86199，OOD mAP@50:95 为 0.53569）。OOD 指标的微降属于典型的域内优化-域外泛化权衡，且被其他十项维度的显著提升所充分补偿。

## 三项改进（多维度协同提升）

Bubble-YOLO11s 的三项改进并非各自为战，而是从**架构、损失函数、数据**三个维度协同作用，共同实现全方位性能提升：

1. **局部对比度细化模块 (P3LCRefine)** — 从**架构维度**出发，约 1,280 参数（仅占 0.013%），插入 Neck P3 层与检测头之间，通过 `x + γ × DWConv(x − AvgPool(x))` 增强小目标局部边缘纹理。主要贡献于**定位精度**和**检测精度**提升。

2. **归一化 Wasserstein 距离损失融合 (NWD Loss)** — 从**损失函数维度**出发，将边框建模为二维高斯分布，以 5:95 比例与 CIoU 融合，提供平滑的小目标定位梯度。主要贡献于**定位精度**提升（mAP@50:95 的相对增幅最大）。

3. **强数据增强配方** — 从**数据维度**出发，Mosaic (p=1.0) + HSV 扰动 + MixUp (p=0.2)，在 644 张训练图上最大化数据多样性。同时贡献于**检测精度**、**定位精度**和**训练稳定性**（种子敏感度范围缩小 2.4×）。

三项改进的叠加效果体现了**多维度涨点**的特征：每一项改进针对不同的性能瓶颈，叠加后形成互补，最终在检测精度、定位精度、训练稳定性等多个维度实现均衡跃升，而参数开销和推理速度几乎不变。
