# 03 — 改进方法

## 概述

本章详细介绍 Bubble-YOLO11s 在 YOLO11s 基线上所做的三项改进。每项改进针对气泡检测的一个核心挑战，三项改进叠加互补，共同实现性能提升。

改进路线图：

```
YOLO11s (COCO预训练, Paper V4数据集)
    │
    ├── 改进一: 局部对比度细化模块 (P3LCRefine)
    │   └── 针对: 气泡小目标、边界模糊
    │
    ├── 改进二: 归一化Wasserstein距离损失融合 (NWD Loss)
    │   └── 针对: 小目标位置偏移的CIoU梯度稀疏
    │
    └── 改进三: 强数据增强配方 (Mosaic + HSV + MixUp)
        └── 针对: 训练数据量有限 (644张)
```

---

## 一、基线模型：YOLO11s

### 1.1 架构概况

YOLO11s 是 Ultralytics 发布的中小型目标检测模型，参数量 9,414,340（约 9.41M），计算量 21.3 GFLOPs（以 768×768 输入计）。

**骨干网络 (Backbone)**：
```
Conv(k=3,s=2,c=64)    → Conv(k=3,s=2,c=128)  → C3k2(c=256,n=2)
→ Conv(k=3,s=2,c=256) → C3k2(c=512,n=2)      → Conv(k=3,s=2,c=512)
→ C3k2(c=512,n=2,e=0.5) → SPPF(k=5,c=512)    → C2PSA(c=512)
```

骨干网络通过 5 次下采样（总 stride=32），输出 P3（80×80×128）、P4（40×40×256）、P5（20×20×512）三个尺度的特征图。C3k2 是 YOLO11 的基础卷积块，SPPF 提供多尺度感受野，C2PSA 引入通道-位置联合注意力。

**特征融合颈部 (Neck)**：
采用 FPN + PAN 双向特征金字塔结构。深层特征经过上采样与浅层特征拼接，再下采样传回深层。P3 特征层（stride=8，80×80）保留了最高空间分辨率，是检测小目标的关键层。

**检测头 (Detect Head)**：
三个检测头分别对应 P3、P4、P5 特征层，输出边界框坐标、目标置信度和类别概率。对于单类（bubble）检测，每个网格单元输出 64 维向量。

### 1.2 预训练与权重重映射

模型使用 MS COCO 数据集（80 类）预训练的 `yolo11s.pt` 权重进行初始化。由于 Bubble-YOLO11s 在 YOLO11s 的拓扑中插入了新模块（P3LCRefine，位于原始模型第 17 层之后），标准权重加载无法直接适���。

采用权重重映射（Weight Remapping）策略：插入点之前的层（0-16 层）直接复制预训练权重；插入点之后的层（原 17-23 层）索引向后偏移一个位置；新增模块（P3LCRefine）采用近恒等映射初始化（见 4.1.2 节）。

## 二、改进一：局部对比度细化模块 (P3LCRefine)

### 2.1 问题分析

气泡检测面临两个核心困难：

1. **目标尺寸小**：气泡中位尺寸约 41-42px（在 768×768 输入中约占 5.5%），属于小目标范畴。P3 特征层（stride=8）是检测此类目标最重要的尺度。
2. **边界模糊**：气泡在显微图像中边界呈渐变半透明状，缺乏清晰的高对比度边缘。标准卷积层在特征提取过程中可能平滑掉这些微弱的局部纹理信息。

在 YOLO11s 的 Neck 结构中，P3 特征图由 P4 上采样后与骨干 P3 跳跃连接拼接再经过卷积得到。这条上采样路径在恢复空间分辨率的同时，可能损失了部分局部高频细节（边缘、纹理）。

### 2.2 模块设计

P3LCRefine（P3 Local Contrast Refinement，P3 局部对比度细化模块）是一个插入在 Neck P3 输出与 Detect P3 输入之间的轻量级模块。

**数据流**：

```
输入: x ∈ R^(B × C × H × W)    (C = 128, H = W = 96)
    │
    ├──→ 平均池化: x_avg = AvgPool_3×3(x)
    │
    ├──→ 对比度提取: contrast = x - x_avg
    │         (减去局部均值，提取高频边缘和纹理信息)
    │
    ├──→ 深度可分离卷积: contrast' = DWConv_3×3(contrast)
    │         (3×3 深度卷积，每个通道独立处理，保持空间局部性)
    │
    └──→ 残差连接: y = x + γ · contrast'
              (γ 为可学习标量，初始化为 1.0)
```

**数学公式**：

```
P3LCRefine(x) = x + γ · DWConv_3×3( x − AvgPool_3×3(x) )
```

其中：
- `AvgPool_3×3(·)`：3×3 窗口、步长 1、边界填充 1 的平均池化
- `DWConv_3×3(·)`：3×3 深度可分离卷积，输入和输出通道数相同
- `γ`：可学习标量参数，初始值 1.0

### 2.3 设计原理

**局部对比度作为先验**：
`x − AvgPool(x)` 操作等价于一个高通滤波器，提取特征图的局部对比度信息。对于气泡检测，这本质上是在增强"边缘"和"纹理"——气泡边界尽管模糊，但仍存在局部亮度变化，该操作能够放大这些微弱信号。

**深度可分离卷积的轻量化**：
标准的 3×3 卷积（输入 C 通道，输出 C 通道）参数量为 9C²。深度可分离卷积分两步：逐通道 3×3 卷积 + 逐点 1×1 卷积，参数量为 9C + C² = C(C+9)。当 C=128 时，标准卷积需 147,456 参数，深度可分离卷积仅需 17,536 参数（减少 88%）。P3LCRefine 的 DWConv 仅保留逐通道部分（未加逐点卷积），参数量进一步降至 9C = 1,152。

**近恒等映射初始化**：
DWConv 的权重初始化为接近零的值（均值为 0，标准差极小），γ 初始化为 1.0。在训练开始时，模块近似恒等映射 `y ≈ x`，不会破坏 COCO 预训练权重已学习到的特征表示。随着训练进行，DWConv 和 γ 逐步学习气泡特定的局部对比度增强模式。

**仅放置在 P3 层**：
P4 和 P5 层检测的目标尺度更大（stride 16 和 32），且感受野更广。气泡中不存在大目标，P4/P5 检测头主要处理中等尺寸气泡（如气泡群）。实验验证（PV4R26, PV4R27）证实，在 P4 或 P5 增加同类模块不带来额外收益。

### 2.4 参数分析

| 属性 | 数值 |
|------|------|
| 参数量 | ~1,280（≈ 0.013% of total 9.41M） |
| 计算增量 | ~0.01 GFLOPs |
| 输入通道数 | 128 |
| 卷积核尺寸 | 3×3 |
| 可学习参数 | γ（标量）+ DWConv 权重（9×128=1,152）+ DWConv 偏置（128） |
| 初始化策略 | γ = 1.0, DWConv weight ≈ 0, bias = 0 |

## 三、改进二：归一化 Wasserstein 距离损失融合 (NWD Loss)

### 3.1 问题分析

标准 YOLO11s 使用 CIoU（Complete IoU）损失作为边界框回归损失：

```
L_CIoU = 1 − IoU + ρ²(b, b^gt) / c² + α · v
```

其中 `ρ²(b, b^gt)` 是预测框与真值框中心点的欧氏距离，`c` 是包围两个框的最小外接矩形的对角线长度，`v` 衡量长宽比一致性，`α` 是动态权重。

**CIoU 对小目标的两个缺陷**：

1. **IoU 对小偏移极度敏感**：对于 40px 的气泡，1 像素的位置偏移即可使 IoU 从 0.95 降至 0.90；3 像素偏移可降至 0.80 以下。这种敏感性导致梯度剧烈波动，不利于小目标的稳定训练。

2. **长宽比项对圆形目标无意义**：气泡近似圆形（w ≈ h），长宽比一致性项 `v` 提供的信息量极少，却引入了额外的噪声梯度。

### 3.2 NWD 原理

归一化 Wasserstein 距离（Normalized Wasserstein Distance, NWD）将边界框建模为二维高斯分布，用 Wasserstein 距离度量分布之间的差异。

**步骤一：边界框→二维高斯分布**

将边界框 `(cx, cy, w, h)` 建模为一个二维高斯分布 N(μ, Σ)：

```
μ = (cx, cy)ᵀ

Σ = [w²/4    0   ]
    [  0    h²/4 ]
```

即假设边界框内的像素质量服从以框中心为均值、半宽半高为标准差的高斯分布。这一建模将离散的边界框坐标转化为连续的概率分布。

**步骤二：Wasserstein 距离**

两个二维高斯分布 N(μ₁, Σ₁) 和 N(μ₂, Σ₂) 之间的 2-Wasserstein 距离为：

```
W² = ‖μ₁ − μ₂‖²₂ + Tr(Σ₁ + Σ₂ − 2(Σ₁^(1/2) Σ₂ Σ₁^(1/2))^(1/2))
```

对于对角协方差矩阵（前述假设），化简为：

```
W² = (cx₁ − cx₂)² + (cy₁ − cy₂)² + (w₁ − w₂)²/4 + (h₁ − h₂)²/4
```

Wasserstein 距离具有两个关键性质：（1）它是一个真正的距离度量（满足三角不等式），（2）它对分布之间的平移和缩放提供连续且平滑的梯度，即使两个框没有重叠区域也能给出有意义的距离。

**步骤三：归一化与损失函数**

```
NWD(Box₁, Box₂) = exp(−W² / C)
L_NWD = 1 − NWD
```

参数 C 是归一化常数，控制距离-相似度映射的"陡峭度"。C 值越小，对位置偏移越敏感；C 值越大，容忍度越高。通过消融实验确定 C = 12.8 为最优值。

**NWD vs CIoU 的梯度特性比较**：

| 特性 | CIoU | NWD |
|------|------|-----|
| 无重叠时的梯度 | 主要来自中心距离项，较小 | 连续平滑，意义明确 |
| 小偏移梯度 | 剧烈变化 | 平滑变化 |
| 尺度敏感性 | 通过 IoU 隐式编码 | 显式分解为位置+尺寸 |
| 长宽比信息 | 显式项 `v` | 通过 Σ 自然编码 |

### 3.3 融合策略

直接将 CIoU 替换为 NWD 会丢失 CIoU 在 COCO 预训练中学到的 IoU 基础定位能力。因此采用**加权融合**策略：

```
L_box = (1 − w_nwd) · L_CIoU + w_nwd · L_NWD
```

其中 w_nwd 通过消融实验确定。

### 3.4 超参数消融

**NWD 融合权重 w_nwd**（固定 C = 12.8）：

| w_nwd | 主测试集综合指标 | 变化 |
|-------|---------------|------|
| 0（仅 CIoU） | 1.23134 | 基线 |
| 0.025 | 1.23712 | +0.00578 |
| **0.05** | **1.23962** | **+0.00828** |
| 0.075 | 1.23891 | −0.00071 |
| 0.10 | 1.23745 | −0.00217 |
| 0.20 | 1.23410 | −0.00552 |

w_nwd = 0.05 为最优，CIoU 与 NWD 以 95:5 的比例融合。过高的 NWD 权重会削弱 CIoU 的精准定位能力，过低则 NWD 的平滑效果不足。

**NWD 归一化常数 C**（固定 w_nwd = 0.05）：

| C | 主测试集综合指标 | 变化 |
|---|---------------|------|
| 6.4 | 1.24273 | −0.02027（显著退化） |
| **12.8** | **1.26300** | 最优 |
| 25.6 | 1.25778 | −0.00522（轻度退化） |

C = 6.4 时，NWD 对小偏移过于敏感，与 CIoU 形成梯度冲突。C = 25.6 时，NWD 过于宽容，平滑效果减弱。C = 12.8 是两个极端之间的最佳平衡点。

## 四、改进三：强数据增强配方

### 4.1 问题分析

Paper V4 训练集仅包含 644 张图像，标签为单一类别（bubble）。原有增强仅包含水平翻转（概率 0.5）和微小平移/缩放（translate=0.02, scale=0.05），数据多样性严重不足。在小数据集上，模型容易过拟合到训练集的具体背景纹理和气泡排列模式，导致泛化性能下降。

早期训练流程中采用保守的增强策略，是因为 V1 数据集已经包含了大量离线增强样本（翻转、旋转、噪声、Copy-Paste），在线增强被设为 0 以避免"信息冗余"。切换到 Paper V4 数据集（仅 644 张图，离线增强仅水平翻转）后，原保守策略成为性能瓶颈。

### 4.2 最终增强配方

| 增强方法 | 参数 | 作用机制 |
|---------|------|---------|
| Mosaic | p = 1.0（1-20 轮），close_mosaic = 10 | 随机选取 4 张训练图，缩放后拼接为一张新图，创造多尺度多背景多样本密度的训练样本 |
| HSV 扰动 | h = 0.015, s = 0.5, v = 0.4 | 随机调整色调、饱和度和亮度，模拟不同光照条件和成像参数下的气泡外观变化 |
| MixUp | p = 0.2, α = β = 32（Beta 分布） | 随机选取两张图像按 Beta 分布权重进行像素级混合，标签也按相同比例混合，软化决策边界 |
| 水平翻转 | p = 0.5 | 增强左右方向的对称不变性 |

### 4.3 Mosaic 增强的关键作用

Mosaic 增强是整体提升中贡献最大的单一因素（+0.01486）。其机制包括：

1. **多尺度多样性**：拼接过程中每张子图独立随机缩放（0.5×-1.5×），单张训练图中包含 4 种不同尺度分布的气泡，迫使模型学习尺度不变特征。

2. **多背景多样性**：拼接的 4 张图来自不同原始图像，可能具有不同的背景纹理（显微镜载玻片的不同区域、不同光照条件）。模型不能依赖特定背景模式来推断目标位置。

3. **密度变化**：4 图拼接意味着每张训练图的目标数量提升为原来的约 4 倍，按实例计数，相当于有效 batch size 扩大 4 倍，有利于损失函数的稳定估计。

4. **上下文多样性**：拼接图的四象限各自独立，Mosaic 构造的合成场景迫使模型更多依赖目标的内部视觉特征（纹理、形状）而非全局上下文线索。

**close_mosaic 机制**：训练最后 10 轮关闭 Mosaic 增强，使用原始（非拼接）图像。这确保模型在训练末期接触真实数据分布，完成从"合成场景"到"真实场景"的分布对齐。

### 4.4 HSV 与 MixUp 的辅助作用

**HSV 扰动**：气泡是半透明的，其外观（亮度、对比度）受背景光照和液体介质的影响很大。HSV 扰动模拟了这一物理变化，增强模型对光照条件的鲁棒性。

**MixUp**：通过软化训练标签（预测目标变为两个图像标签的加权混合），MixUp 降低了模型对单个训练样本的过拟合程度，使决策边界更加平滑。

### 4.5 增强消融叠加

| 配置 | 主测试集综合指标 | 增益 |
|------|---------------|------|
| Baseline (COCO-init + P3LCRefine + NWD) | 1.23962 | — |
| + Mosaic (p=0.5) + HSV | 1.25141 | +0.01179 |
| + Mosaic (p=1.0) | 1.25448 | +0.00307 |
| + MixUp (p=0.2) | 1.25554 | +0.00106 |

Mosaic 是增益最大的单一项（+0.01486 从无增强到 Mosaic=1.0）。MixUp 的边际增益虽小（+0.00106），但在已饱和的管道中仍是正向贡献。

### 4.6 未纳入的增强方法

以下增强方法经过实验验证，未带来正向收益：

- **Copy-Paste**（p=0.3）：V1 数据集已有离线 Copy-Paste 增强，在线叠加无效
- **CutMix**（p=0.5）：与 Mosaic 的功能重叠（区域混合 vs 拼接混合）
- **Mosaic-9**（9 图拼接）：图块过小导致气泡缩小到不可分辨
- **随机旋转**（±5°）：破坏了气泡的自然空间排列模式

### 4.7 增强管道饱和

在达到最优配方后，进一步增强（增加增强强度、引入新增强方法）均未带来正向收益。634 张训练图的样本多样性已接近信息上限，增强管道饱和。进一步的性能提升需要从其他方向突破（如架构改进、损失函数优化、更大的基础数据集）。

## 五、改进效果总览：全方位多维度涨点

本章所述的三项改进——P3LCRefine（架构维度）、NWD 损失融合（损失函数维度）、强数据增强配方（数据维度）——并非各自孤立，而是从三个互补方向共同发力，实现了**检测精度、定位精度、参数效率、计算效率、训练稳定性的全方位提升**。

### 5.3 各改进贡献占比与维度解读

（以总综合指标增益 +0.04793 为分母）：

| 改进 | 增益 | 占比 | 主要贡献维度 |
|------|------|------|------------|
| P3LCRefine | +0.01627 | 33.9% | 检测精度、定位精度 |
| Mosaic + HSV + MixUp | +0.01592 | 33.2% | 检测精度、定位精度、**训练稳定性** |
| NWD Loss | +0.00828 | 17.3% | **定位精度**（mAP@50:95 提升最大） |
| 种子选择 | +0.00746 | 15.6% | 综合性能峰值 |

三项核心改进的贡献大致均衡（约 34% / 33% / 17%），不存在单一改进"主导"全局的情况，印证了三项改进形成了有效的**多维度互补**。

### 5.4 多维度提升的关键解读

**（1）定位精度涨幅最大，击中核心痛点**

mAP@50:95 的相对提升（+4.94%）显著高于 mAP@50（+3.42%），这一差距并非偶然。NWD 损失融合将边界框建模为二维高斯分布，以 Wasserstein 距离替代 IoU 进行相似度度量，为小目标提供了连续平滑的定位梯度——直接作用于边界框回归质量。P3LCRefine 通过增强 P3 层的局部边缘纹理，为检测头提供了更清晰的空间特征——间接改善了定位精度。数据增强（尤其是 Mosaic）通过多尺度拼接增加了目标尺度多样性，使模型对不同尺度的气泡都能产出精确的边界框。三个方向虽然出发点不同，但**合力指向了"提升定位精度"这一共同目标**。

**（2）计算效率零代价，部署无需妥协**

P3LCRefine 模块仅增加了约 1,280 个参数（0.013%），其核心操作——平均池化和深度可分离卷积——均为高度优化的标准算子，不引入任何计算瓶颈。NWD 损失在推理时完全不需要计算（仅训练时使用）。数据增强完全不改变模型结构。因此，Bubble-YOLO11s 的推理速度与 YOLO11s 完全相同，可直接替换部署而无需升级硬件或调整推理流水线。这种**"零代价涨点"**的特性是该改进方案工程实用性的集中体现。

**（3）训练稳定性显著改善，增强工程可复现性**

旧配方下不同随机种子的性能差异可达 0.01814（以综合指标 S 计）；引入 Mosaic 强增强后，两个不同种子的差异缩小至 0.00746——稳定性提升了约 2.4 倍。这一改善并非以牺牲峰值性能为代价（种子 48 在旧配方下反而是三个种子中最差的，但在新配方下成为最优），而是强增强通过增加每轮训练数据的多样性和随机性，使得不同初始化的模型都能更好地收敛到高质量的解空间。对于工程实践而言，这意味着模型训练结果更加可复现，减少了对"幸运种子"的依赖。

### 5.5 OOD 泛化与域内性能的平衡

改进模型在分布外测试集上的表现体现了域内优化与跨域泛化之间的经典权衡：

| 指标 | 基线 (OOD) | Bubble-YOLO11s (OOD) |
|------|-----------|---------------------|
| OOD mAP@50 | 0.87396 | 0.86199 |
| OOD mAP@50:95 | 0.55276 | 0.53569 |

OOD 指标较基线微有下降，但绝对性能依然保持在较高水平。值得强调的是：OOD 微降集中体现在 mAP@50 上（检测召回率），而 mAP@50:95 的降幅相对更小，说明即使在跨域场景下，模型的定位精度仍然受到 NWD 损失和 P3LCRefine 的正面影响。OOD 性能的这一小幅波动，被检测精度、定位精度、参数效率、计算效率和训练稳定性的全面提升所充分补偿——这是一个典型的多维度全局优化结果。
