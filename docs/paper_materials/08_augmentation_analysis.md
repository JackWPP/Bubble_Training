# 08 — 数据增强分析

## 增强探索全景

本项目系统性地探索了 7 种在线数据增强方法：

| 增强 | 参数 | 效果 (Δ main sum) | 结论 |
|------|------|------------------|------|
| **Mosaic** | p=0.5→1.0 | **+0.01486** | 最关键突破 |
| **HSV 抖动** | h=0.015, s=0.5, v=0.4 | 包含在 Mosaic 配方中 | 重要组成部分 |
| **MixUp** | p=0.2 | +0.00106 (seed44) | 微弱正收益 |
| Copy-Paste | p=0.3 (flip) | ±0 | 与已有 flip 重复 |
| CutMix | p=0.5 | ±0 | 无额外收益 |
| Mosaic9 | p=1.0 (n=9) | ±0 | 无额外收益 |
| Degrees | 5° | -0.01157 | 有害 |

## 为什么 Mosaic 如此关键

### 1. 之前为何 mosaic=0？

早期的训练配置继承了一个保守策略：因为数据已经做了离线增强，所以禁用所有在线增强以避免"过度增强"。但 Paper V4 的离线增强只有水平翻转——这远不足以覆盖数据多样性需求。

### 2. Mosaic 创造了什么？

Mosaic 将 4 张随机采样的图片拼接为 1 张 768×768 的训练样本：

```
┌──────────┬──────────┐
│  Image 1 │  Image 2 │
│ (crops)  │ (crops)  │
├──────────┼──────────┤
│  Image 3 │  Image 4 │
│ (crops)  │ (crops)  │
└──────────┴──────────┘
      768 × 768
```

每张样本包含 4 个不同场景、不同光照、不同密度的气泡。这大大增加了每个 batch 中的**数据多样性**和**场景组合数**。

### 3. 定量效果

| Mosaic 概率 | main mAP50 | main mAP50-95 | main sum |
|------------|-----------|-------------|---------|
| 0 (关闭) | 0.81210 | 0.42752 | 1.23962 |
| 0.5 | 0.81363 | 0.43779 | 1.25141 |
| 1.0 | 0.81379 | 0.44070 | 1.25448 |

主要增益在 mAP50-95（+0.01318），说明 Mosaic 主要改善了定位精度而非检测召回。

## 增强管道饱和

从 Mosaic=0.5 到最终配方（+MixUp, +CutMix, +Mosaic9, +Degrees），额外收益递减至零或负值。这说明：

1. **644 张训练图所能支持的数据多样性已接近上限**
2. 进一步的数据增强应通过**采集更多真实数据**而非合成增强
3. 增强管道的边际效用递减与训练集大小正相关

## 增强策略建议

对于类似的小样本目标检测任务：
1. **优先开启 Mosaic**（即使数据已有离线增强）
2. **HSV 抖动**对光照变化大的场景（如气泡、水下、医疗影像）特别重要
3. **MixUp 可加可不加**（收益在 0.1% 级别）
4. **不需要 Mosaic9 / CutMix / Copy-Paste**（边际收益为零）
5. **不推荐旋转**（可能引入不符合物理规律的样本）
