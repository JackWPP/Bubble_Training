# 02 — 改进方法论

## Bubble-YOLO11s: YOLO11s + 三项改进

Bubble-YOLO11s 在标准 YOLO11s 基础上引入三项改进，分别针对气泡检测的三个核心挑战：

### 挑战 1: 气泡是小目标，边界模糊
→ **改进: P3LCRefine（P3 局部对比度细化模块）**

### 挑战 2: 标准 CIoU loss 对小目标位置偏移过于敏感
→ **改进: NWD（归一化 Wasserstein 距离）loss 融合**

### 挑战 3: 数据量有限（644 张训练图），模型泛化不足
→ **改进: Mosaic + HSV + MixUp 强数据增强配方**

---

## 改进一: P3LCRefine

### 动机
气泡是典型的小目标（平均尺寸 ~47px），在高分辨率特征图（P3 层，stride=8）上检测。原始的 P3 特征图来自骨干网络的上采样路径，缺乏足够的局部细节信息。

### 设计
P3LCRefine 是一个**轻量级局部对比度细化模块**，插入在 YOLO11s 颈部 P3 特征图与检测头之间。

```
输入 x (P3 特征图, C=128, H=96, W=96)
    │
    ├──→ 局部对比度: c = x - AvgPool(x)
    │
    ├──→ 深度卷积: c' = DepthwiseConv3×3(c)
    │
    ├──→ gamma 缩放: c'' = γ · c'     (γ 初始化为 1.0，可学习)
    │
    └──→ 残差连接: y = x + c''
```

- **参数量**: ~1.2K（可忽略）
- **计算量**: ~0.01 GFLOPs
- **效果**: 增强局部边缘和纹理细节，改善小目标定位

### 为何只放在 P3？
P4 和 P5 检测更大的目标（虽然有气泡数据中没有大目标），添加 P4/P5 模块会增加参数但没有对应收益。实验验证（PV4R26, PV4R27）也确认了这一点。

## 改进二: NWD Loss 融合

### 动机
标准 CIoU loss 使用 IoU + 中心距离 + 长宽比三项。对于小目标：
- IoU 对位置偏移极度敏感（即使 1px 偏移就能导致 IoU 大幅下降）
- 长宽比项对近圆形气泡无意义

### NWD 原理
NWD（Normalized Wasserstein Distance）将边界框建模为 2D 高斯分布，用 Wasserstein 距离度量两个分布之间的差异：

```
Wasserstein² = (cx₁-cx₂)² + (cy₁-cy₂)² + (w₁-w₂)²/4 + (h₁-h₂)²/4
NWD = exp(-Wasserstein² / C)
```

其中 C=12.8 是一个归一化常数。

### 融合方式
```
L_box = (1 - w_nwd) × L_CIoU + w_nwd × (1 - NWD)
```

- w_nwd=0.05 为最优融合权重（实验验证）
- NWD 提供平滑的距离度量，对微小位置偏移更宽容
- CIoU 保留 IoU-based 的精准定位能力

### 效果
NWD 融合使 main sum 从 1.23134 提升到 1.23962（+0.00828）。

## 改进三: 强数据增强配方

### 动机
Paper V4 训练集仅 644 张图片，原有增强仅水平翻转。增强不足导致模型泛化能力受限。

### 增强配方

| 增强 | 参数 | 作用 |
|------|------|------|
| **Mosaic** | p=1.0, 前 20 epoch | 将 4 张图拼接为 1 张，创造多尺度多背景训练样本 |
| **HSV 抖动** | h=0.015, s=0.5, v=0.4 | 模拟不同光照和色彩条件（气泡半透明） |
| **MixUp** | p=0.2 | 两张图按 Beta(32,32) 权重混合，软化决策边界 |
| **水平翻转** | p=0.5 | 对称不变性 |
| **平移+缩放** | translate=0.02, scale=0.05 | 位置和尺度鲁棒性 |

### Mosaic 为何至关重要
Mosaic 将 4 张不同图片拼接为一张新的训练样本。这创造了：
1. **多尺度**: 拼接后目标相对尺寸变化更大
2. **多背景**: 同一张图包含 4 种不同背景
3. **密集场景**: 拼接使目标密度增加（训练时 Instance 数更多）

之前所有实验都在 mosaic=0 条件下进行，这严重限制了数据多样性。

### 效果
Mosaic+HSV+MixUp 配方使 main sum 从 1.23962 提升到 1.25554（+0.01592）。

## 改进效果汇总

| 改进 | 累积 main sum | 单项增益 |
|------|-------------|---------|
| Baseline (YOLO11s + COCO-init) | 1.21507 | — |
| + P3LCRefine + NWD | 1.23962 | +0.02455 |
| + Mosaic(0.5) + HSV | 1.25141 | +0.01179 |
| + Mosaic(1.0) | 1.25448 | +0.00307 |

| + Seed48 (最优种子) | **1.26300** | **+0.00746** |
