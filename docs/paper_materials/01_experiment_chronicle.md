# 01 — 实验编年史

## 概述

本项目旨在改进 YOLO11s 在气泡检测任务上的性能。基线模型使用 COCO 预训练权重，在 Paper V4 数据集（768×768）上训练 30 epoch，达到 main sum = 1.21507（mAP50=0.79803, mAP50-95=0.41704）。

经过三轮共 18 个实验，最终最佳模型达到 main sum = **1.26300**（mAP50=0.82535, mAP50-95=0.43765），相对基线提升 **+3.94%**。

---

## 提升路径

```
Baseline (YOLO11s + COCO-init):        1.21507
    ↓
+P3LCRefine + NWD W=0.05:              1.23962  (+0.02455)  ← 接手时最佳
    ↓
+Mosaic=0.5 + HSV:                      1.25141  (+0.01179)  ← Mosaic 突破
    ↓
+Mosaic=1.0:                            1.25448  (+0.00307)
    ↓
+MixUp:                                 1.25554  (+0.00106)  ← 增强配方完善
    ↓
Seed48 (相同配方):                      1.26300  (+0.00746)  ← 最终最佳
```

总提升：**+0.04793（+3.94% over baseline）**

---

## 实验轮次

### Round 1: 快速排查（3 个实验）

目标：验证一些低成本假设，快速排除无效方向。

| 实验 | 基线 | 结果 | 结论 |
|------|------|------|------|
| NMS IoU=0.60 | W01_seed44=1.23882 | 1.24257 (+0.003) | 免费但收益太小 |
| Copy-Paste (flip, p=0.3) | W01_seed44=1.23882 | 1.23882 (±0) | 与已有 flip 增强重复 |
| WIoU v3 loss | W01_seed44=1.23882 | 1.22108 (-0.018) | 不如 CIoU，排除 |

### Round 2: Mosaic 突破（9 个实验）

目标：发现并优化数据增强配方。

| 实验 | 配置 | 结果 | 收益 |
|------|------|------|------|
| W01 + Mosaic=0.5 + HSV | 首次开启 Mosaic | 1.24618 | +0.00656 |
| W005 + Mosaic=0.5 + HSV | W005 权重复现 | 1.25141 | +0.00523 |
| W005 + Mosaic=1.0 + HSV | 100% Mosaic | 1.25448 | +0.00307 |
| W005 + Mosaic=1.0 + HSV + MixUp | 加 MixUp | 1.25554 | +0.00106 |
| W005 + Mosaic=1.0 + ... + AdamW | AdamW 优化器 | 1.24039 | -0.01515 |
| W005 + Mosaic9 + HSV + MixUp | 9 图拼接 | 1.25554 | ±0 |
| W005 + Mosaic=1.0 + ... + E50 | 50 epoch | 1.25039 | -0.00409 |
| TTA (Test-Time Aug) | 推理增强 | 0.79807 | -0.019 |
| W005 + Mosaic=1.0 + ... + Seed48 | 换种子 | **1.26300** | **+0.00746** |

**关键发现：Mosaic=0 → Mosaic=1.0 是最大的单次提升（+0.01486）。** Seed48 在这个配方下显著优于 seed44。

### Round 3: 精细调优（6 个实验）

目标：在最优配方上微调超参数，榨取最后收益。

| 实验 | 配置 | 结果 | 结论 |
|------|------|------|------|
| +CutMix (p=0.5) | 额外增强 | 1.26300 (±0) | 无额外收益 |
| +Degrees=5 | 旋转增强 | 1.25143 (-0.012) | 有害 |
| NWD C=6.4 | 更严格 NWD | 1.24273 (-0.020) | 严重退化 |
| NWD C=25.6 | 更宽松 NWD | 1.25778 (-0.005) | 轻微退化 |
| close_mosaic=5 | 更短适应期 | 1.26300 (±0) | 无影响 |
| warmup_epochs=6 | 更长预热 | 1.24869 (-0.014) | 有害 |

**结论：C=12.8, close_mosaic=10, warmup=3 已是最优。增强管道已饱和。**
