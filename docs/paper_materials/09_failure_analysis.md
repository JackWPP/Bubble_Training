# 09 — 负结果分析

## 概述

在探索过程中，多种方向未能产生预期收益。这些负结果同样具有学术价值，可帮助后续研究者避免重复无效尝试。

## 失败方向汇总

### 1. WIoU v3 Loss（-0.018）

**假设**: WIoU v3 的动态非单调聚焦机制对小目标定位优于 CIoU

**结果**: main sum 从 1.23882 降至 1.22108

**分析**: WIoU v3 的去长宽比惩罚项虽然适合圆形气泡，但其离群度检测机制（β = L_IoU / mean(L_IoU)）依赖于足够的 batch 统计。对于单类检测+小 batch（8），EMA 追踪的 mean_L_IoU 可能不稳定。此外，CIoU 已经在 COCO 预训练中得到了充分优化。

### 2. AdamW 优化器（-0.015）

**假设**: AdamW 的自适应学习率能更好地处理 Mosaic 带来的强增强噪声

**结果**: main sum 从 1.25554 降至 1.24039

**分析**: SGD+Momentum 的隐式正则化效应在小数据集上优于 AdamW 的显式自适应。与文献中"AdamW 在小 batch 训练中不如 SGD"的结论一致。

### 3. TTA 测试时增强（-0.019）

**假设**: 推理时的水平翻转+多尺度测试能提升检测精度

**结果**: mAP50 从 0.817 降至 0.798

**分析**: TTA 产生更多候选框，但在密集气泡场景中引入了更多误检（precision 从 0.797 降至 0.779）。对于单类密集小目标，TTA 的 NMS 合并策略不够优化。

### 4. Degrees（旋转增强, -0.012）

**假设**: 气泡无固定方向，小角度旋转增强合理

**结果**: main sum 从 1.26300 降至 1.25143

**分析**: 虽然气泡确实无方向性，但气泡群的整体空间排列可能不满足旋转不变性。旋转后气泡之间的空间关系被破坏，可能引入了不符合物理规律的训练样本。

### 5. E50 延长训练（-0.004）

**假设**: 更多 epoch 能进一步提升性能

**结果**: 最佳 epoch 在 14，30 epoch 后开始过拟合

**分析**: 小数据集（644 张）不支持更长的训练。Early Stopping 在 epoch 44 触发（patience=30）。即使配合强增强，模型容量（9.4M 参数）相对训练集大小仍有过拟合风险。

### 6. NWD C=6.4（-0.020）

**假设**: 更小的 NWD 常数使距离度量更严格，适合小气泡

**结果**: 显著退化

**分析**: C=6.4 使 NWD 对位置偏移过度敏感，产生的梯度信号与 CIoU 冲突，反而阻碍了收敛。

## 教训总结

| 经验 | 适用场景 |
|------|---------|
| CIoU + NWD 优于其他 IoU 变体 | 小目标检测 |
| SGD > AdamW | 小 batch + 小数据集 |
| TTA 不一定有益 | 密集单类检测 |
| 旋转增强需谨慎 | 目标存在空间排列规律时 |
| 30 epoch 是小数据集甜区 | ~600 张训练图 |
| NWD 常数 12.8 具有通用性 | 不同目标尺寸分布 |
