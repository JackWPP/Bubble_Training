# 06 — OOD 泛化分析

## OOD 数据集

OOD（Out-of-Distribution）数据集 `yolo_dataset_grouped` 包含不同的气泡场景，用于评估模型在训练分布外的泛化能力。

| 数据集 | 图片数 | 实例数 | 特点 |
|--------|-------|--------|------|
| Paper V4 (train) | 644 | ~22,000 | 训练分布 |
| Paper V4 (test) | 83 | 2,724 | 同分布测试 |
| Grouped (test) | 208 | 6,108 | OOD，不同场景/密度 |

## OOD 性能对比

基于 W005 + Mosaic(1.0) + HSV + MixUp E30 配方：

| 模型 | Main mAP50 | OOD mAP50 | OOD mAP50-95 | OOD vs Main Gap |
|------|-----------|----------|-------------|----------------|
| Baseline (W005, no Mosaic) | 0.81210 | 0.85363 | 0.52379 | +0.04153 |
| + Mosaic(0.5) + HSV | 0.81363 | 0.87656 | 0.56906 | +0.06293 |
| + Mosaic(1.0) | 0.81379 | 0.86776 | 0.55433 | +0.05397 |
| + MixUp (seed44) | 0.81712 | 0.85368 | 0.53439 | +0.03656 |
| + MixUp (seed48) | 0.82535 | 0.86199 | 0.53569 | +0.03664 |

## 关键发现

### 1. OOD mAP50 始终高于 Main mAP50
OOD 数据集中气泡可能更大、更稀疏，检测难度更低。这是合理的 domain gap。

### 2. Mosaic+HSV 大幅提升 OOD 泛化
Mosaic(0.5)+HSV 使 OOD mAP50-95 从 0.52379 提升到 0.56906（+8.6%）。Mosaic 创造的多场景训练样本直接提升了模型对不同数据分布的适应能力。

### 3. Mosaic=1.0 降低 OOD 但提升 Main
Mosaic=1.0 使 OOD mAP50-95 从 0.56906 降到 0.55433（-2.6%），但 Main mAP50-95 从 0.43779 升到 0.44070。存在一定的 main/OOD trade-off。

### 4. MixUp 轻微降低 OOD
在 seed44 上，MixUp 使 OOD mAP50-95 从 0.55433 降到 0.53439，但 Main mAP50 从 0.81379 升到 0.81712。这表明 MixUp 的软标签混合可能使模型更适配上训练分布，但略微降低了极端泛化能力。

## OOD 最强模型

按 OOD mAP50-95 排序：

| 模型 | OOD mAP50-95 | Main mAP50 | 备注 |
|------|-------------|-----------|------|
| W005 + Mosaic(0.5) + HSV | **0.56906** | 0.81363 | OOD 最佳 |
| W01 + NWD seed48 | 0.58607 | 0.79975 | 前期实验 |
| W005 + Mosaic(1.0) + HSV | 0.55433 | 0.81379 | Main/OOD 平衡 |
| W005 + MixUp seed48 | 0.53569 | 0.82535 | Main 最佳 |

如果应用场景优先考虑 OOD 鲁棒性，推荐 Mosaic(0.5) 配方。如果优先 main test 性能，推荐 Mosaic(1.0)+MixUp+seed48。
